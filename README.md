## ❓ Ответы на контрольные вопросы

### 1. В чём разница между map() и forEach()? Когда использовать каждый?
* **forEach()** используется, когда нужно просто перебрать массив и выполнить какое-то действие для каждого элемента (например, отрисовать карточки в HTML). Он ничего не возвращает (`undefined`).
* **map()** используется, когда нужно трансформировать данные. Он перебирает массив, изменяет каждый элемент по заданному правилу и **возвращает абсолютно новый массив** такого же размера, не меняя исходный.

### 2. Что возвращает filter()? Изменяет ли он исходный массив?
Метод `filter()` возвращает **новый массив**, в который попадают только те элементы исходного массива, которые прошли заданную проверку (колбэк-функция вернула `true`). Исходный массив при этом остается неизменным.

### 3. Что такое JSON? Чем JSON-строка отличается от JavaScript-объекта?
**JSON** (JavaScript Object Notation) — это строгий текстовый формат обмена данными. 
Главное отличие в том, что JS-объект «живет» в оперативной памяти браузера, он динамический и к его свойствам можно обращаться через точку (`product.price`). JSON — это просто «мертвая» строка текста, которую удобно передавать по сети или хранить в `localStorage`.

### 4. Что произойдёт, если передать в JSON.parse() невалидную строку?
Программа упадет с ошибкой `SyntaxError: Unexpected token`. Выполнение скрипта на этой строчке остановится, и код, написанный ниже, не сработает.

### 5. Что такое деструктуризация объекта? Приведите пример из своего кода.
Деструктуризация — это удобный синтаксис, который позволяет «распаковать» нужные свойства объекта в отдельные переменные в одну строчку. 
*Пример:* `const { name, price } = product;` (вместо того, чтобы писать `const name = product.name; const price = product.price;`).

### 6. Что делает оператор spread (...) при копировании объекта?
Оператор `...` (spread) «рассыпает» все свойства старого объекта в новый. В моем коде он использовался внутри `map()`: `{ ...product, label: "..." }`. Это позволило скопировать все старые данные товара и безопасно добавить к ним новое свойство `label`.

### 7. Почему нельзя сравнивать объекты через ===? Что выведет: {} === {} ?
Выведет `false`. Объекты (и массивы) — это ссылочный тип данных. Переменная хранит не сами данные объекта, а ссылку (адрес в памяти) на него. Два разных объекта `{}`, даже если они абсолютно пустые, лежат по разным адресам в памяти, поэтому они не равны.

### 8. Что такое метод reduce()? Как вы использовали его для подсчёта суммы?
Метод `reduce()` сворачивает массив к одному единственному значению. Он проходит по каждому элементу и накапливает результат в специальную переменную. 
В проекте я использовала его для подсчета общей суммы корзины: `prices.reduce((sum, price) => sum + price, 0)`, где `0` — это стартовое значение суммы, к которому по очереди прибавлялась каждая цена.
